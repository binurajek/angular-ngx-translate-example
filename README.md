# Angular i18n Dashboard Demo

This is a demonstration project showing how to implement internationalization (i18n) in an Angular application using `@ngx-translate`.

It features a responsive, glassmorphism-styled dashboard that can switch between English, Dutch, and French on the fly without reloading the page.

## Setup Instructions

Follow these steps to set up and run the project locally:

1. Make sure you have Node.js and npm installed on your system.
2. Open your terminal and navigate to the project directory: `cd angular-i18n` (if you are not already inside it).
3. Install the required dependencies by running the following command:
   `npm install`
4. Once the dependencies are installed, start the local development server by running:
   `npm start` or `ng serve`
5. Open your web browser and navigate to `http://localhost:4200/`.
6. You will see the dummy dashboard. Use the language dropdown in the top right corner of the header to switch between English, Dutch, and French. The text on the page will translate automatically.

## Running Tests

To run the unit tests for this project, run the following command in your terminal:

`npm test` or `ng test`

## Build for Production

To build the project for production, run the following command:

`npm run build` or `ng build`

The build artifacts will be stored in the `dist/` directory.
